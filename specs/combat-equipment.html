01 Vue d'ensemble

EriniumFaction introduit trois minerais progressifs au-dessus du diamant, chacun avec sa propre couleur dominante, ses materiaux bruts et raffines, ses outils et armures complets. La progression naturelle du joueur va du Valterite (+0% vs diamant) a l'Aetherite (+15%) puis a l'Erinium (+35%).

08 Generation du monde

Les trois minerais sont generes dans l'Overworld via IWorldGenerator. La rarete est relative : Valterite > Aetherite > Erinium, avec l'Erinium tres rare et profond.

Distribution

Minerai Y min Y max Taille veine Veines / chunk Rarete relative
Valterite 5 40 4-8 2-4 Peu commun
Aetherite 5 25 2-5 1-2 Rare
Erinium 1 15 1-3 0-1 Tres rare

Regles de generation

  • Dimension — Overworld uniquement (dimension ID 0). Pas de generation dans le Nether ou l'End.
  • Biomes exclus — Aucun biome exclu par defaut (configurable via config future).
  • Chunk seed — Utilise la seed du chunk pour garantir la reproducibilite avec la meme seed de monde.
  • Erinium 0-1 veine — La valeur "0" signifie qu'il peut ne pas apparaitre dans un chunk. Le calcul utilise un Random par chunk pour le nombre de veines.

Implementation

@GameRegistry.worldGenerator public class EriniumWorldGenerator implements IWorldGenerator { @Override public void generate(Random rand, int chunkX, int chunkZ, World world, IChunkGenerator gen, IChunkProvider prov) { if (world.provider.getDimension() != 0) return; if (!FactionConfig.oreGenerationEnabled) return; generateOre(rand, world, chunkX, chunkZ, EriniumBlocks.VALTERITE_ORE.getDefaultState(), FactionConfig.valteriteVeinSize, FactionConfig.valteriteVeinsPerChunk, 5, FactionConfig.valteriteMaxHeight); // idem pour aetherite et erinium... } private void generateOre(...) { new WorldGenMinable(state, size).generate(world, rand, world.getTopSolidOrLiquidBlock(...)); } }

12 Equilibre et design

Philosophie PvP

  • La progression doit etre significative mais pas ecrasante. Un joueur en Erinium ne doit pas one-shot un joueur en Aetherite — le skill reste determinant.
  • Les degats bonus sont calcules en DPS reel (degats * vitesse). L'Erinium donne environ +35% de DPS vs diamant, pas 35% de degats bruts.
  • La durabilite plus elevee recompense l'investissement dans la progression sans rendre l'equipement jetable.

Progression naturelle

  • Un nouveau joueur trouve du Valterite avant de pouvoir crafter le diamant necessaire pour miner l'Aetherite.
  • L'Erinium est intentionnellement rare — un set complet est un objectif de mid/late game pour une faction etablie.
  • Les blocs de stockage permettent un stockage compact pour les grandes factions qui accumulent des ressources.

Compatibilite future

  • Les ToolMaterial et ArmorMaterial sont definis comme constantes statiques dans EriniumMaterials pour faciliter l'extension future (nouveaux items).
  • La config de worldgen laisse ouverte la possibilite d'ajouter de nouveaux minerais sans modifier le generateur principal.
  • Les armures pourront recevoir des effets de set bonus dans une version ulterieure (via mixin EntityPlayer ou capability).
